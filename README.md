# 🌟 Site Paroquial Rainha da Paz

Este é o repositório do site oficial da **Paróquia Rainha da Paz**, desenvolvido para disponibilizar conteúdos religiosos, informações da comunidade e facilitar o acesso dos fiéis a recursos espirituais no dia a dia.

O site reúne páginas dedicadas a orações, leitura diária, dízimo e conteúdos pastorais, com uma navegação simples e acessível.

## 🚀 Funcionalidades

* **Leitura Diária:** Acesso a conteúdos religiosos para reflexão diária.
* **Orações:** Página com diversas orações para diferentes momentos.
* **Dízimo:** Informações e acesso à contribuição da comunidade.
* **Pastoral da Catequese:** Conteúdos e informações voltadas à formação cristã.
* **Interface Simples:** Navegação intuitiva e leve, focada na acessibilidade.

## 🛠️ Tecnologias Utilizadas

* **Front-end:** HTML5
* **Estilização:** CSS3
* **Interatividade:** JavaScript

## 📁 Estrutura do Projeto

* `index.html` → Página inicial do site
* `leitura-diaria.html` → Conteúdo de leitura diária
* `oracoes.html` → Lista de orações
* `dizimo.html` → Página de dízimo
* `pascom.html` → Página da PASCOM
* `pastoraldacatequese.html` → Conteúdos da catequese

## ⚙️ Como rodar o projeto

1. Clone o repositório:

```bash
git clone https://github.com/fssamuelbs/siteparoquiarainhadapaz.git
```

2. Acesse a pasta do projeto:

```bash
cd siteparoquiarainhadapaz
```

3. Abra o arquivo no navegador:

* Basta abrir o `index.html`
